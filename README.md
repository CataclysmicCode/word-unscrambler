# word-unscrambler

created by Cataclysmic code 5/7/2023

  This program takes eight letters (or less) and turns them into a words by first finding all the possible conbinations of the letters and then for every combination, finding all the real word in the dictionary that exist in those combinations.

  Depending on what results you want, you may want to change the dictionary you are using from the default one I set.
